# CtfReg Cog

A Red Discord Bot cog for managing CTF contest registration using CTFTime integration.

## Installation

### Via Discord (Recommended)

```
[p]repo add vcth4nh-cogs https://github.com/vcth4nh/vcth4nh-cogs
[p]repo install vcth4nh-cogs
[p]load ctfreg
```

### Manual Installation

1. Navigate to your Red's cogs directory:
   ```bash
   cd ~/.local/share/Red-DiscordBot/cogs
   ```

2. Clone the repository:
   ```bash
   git clone https://github.com/vcth4nh/vcth4nh-cogs.git
   ```

3. Load the cog in Red:
   ```
   /load ctfreg
   ```
